# employees 表结构

| 列名       | 类型    | 说明                   |
|------------|---------|------------------------|
| id         | INTEGER | 主键                   |
| name       | TEXT    | 员工姓名               |
| department | TEXT    | 部门名称               |
| salary     | INTEGER | 月薪（元）             |
| hire_date  | TEXT    | 入职日期 YYYY-MM-DD    |

## 已有部门
技术部、市场部、人事部、财务部、产品部
